Security Overview

Phishing Detection

Phishing a type of cyber attack where attackers trick individuals into divulging sensitive information, such as usernames, passwords, and credit card details, by posing as a trustworthy entity. Our phishing detection system employs machine learning algorithms to analyze URLs and assess their likelihood of being phishing attempts. Key features include:

  • URL analysis using TF-IDF (Term Frequency-Inverse Document Frequency) techniques
  • Random Forest classification for predicting phishing URLs
  • Real-time assessment of user-submitted URLs

Spam Text Detection

Spam texts, often delivered via email, are unsolicited messages sent in bulk, typically for advertising or phishing purposes. Our spam text detection system utilizes machine learning to classify emails as either "ham" (not spam) or "spam." Key features include:

  • Dataset training with a collection of spam and non-spam (ham) emails
  • CountVectorizer and Naive Bayes algorithm for email text classification
  • Effective filtering of unwanted messages from user inboxes

Password Security with SHA Key

Password security is a critical aspect of protecting user accounts. Our system enhances password security using SHA hashing. When users enter a password, its SHA-1 hash is generated and checked against a database of compromised passwords. Key features include:

  • SHA-1 hashing for password encryption
  • Integration with the "Have I Been Pwned" API to check for compromised passwords
  • Real-time feedback on password security status

Disclaimer

This security system provides an additional layer of protection but does not guarantee absolute security. Users are encouraged to follow best practices for online security, including the use of strong, unique passwords and vigilance against phishing attempts.
